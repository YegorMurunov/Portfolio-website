How the development stage works?

If it's an html layout from scratch, I use my Gulp+Webpack build. In it I use: html, scss, js, webp (Modern Image Format), file minimization, image compression, and many more small goodies.

And if it's a refinement of the html layout, I use the stack that was used before me. And before I start writing code, I analyze it to write code in the same style.

If the development takes place on the react library, then I use the stack that is currently used in the project. Or, if it's a project from scratch, I discuss my stack with the customer.

And most importantly, I write legible, understandable, and highly scalable code!
